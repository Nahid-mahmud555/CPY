What is CPY

One bubble, four file formats, zero servers.

CPY is a lightweight extension that lives quietly on every page. Highlight any text — an AI answer, a paragraph from an article, a research quote — and a small bubble appears. One click saves it to your browser's local storage.

Whenever you're ready, open the popup and export your entire collection as a clean file. Everything runs on-device: nothing is ever uploaded, logged, or shared.

.json .csv .pdf .txt
  • 💾
    Local-first — data never leaves your device
  • 🧼
    Auto-cleaned — line breaks & duplicates handled for you
  • 🌐
    Cross-browser — one codebase, Chrome & Firefox
  • 🔒
    No tracking — no analytics, no external requests

Compatibility

Where CPY works — and where it can't

The exceptions aren't bugs — browsers deliberately block every extension from these areas.

✅ Works great

  • Any website — blogs, docs, news, articles
  • AI chat platforms — ChatGPT, Gemini, Claude & similar
  • Any http/https page with real, selectable text

❌ Can't work here

  • Browser's built-in PDF viewer — a privileged internal page
  • chrome:// / about: / view-source: — reserved browser UI
  • Google Docs / Sheets / Slides — renders text on canvas, not HTML
